# 유닉스 / 리눅스 명령어

정보처리기사 실기에서 자주 나오는 **유닉스·리눅스 기본 명령어**를 용도별로 정리했습니다.

---

## 1. 파일·디렉터리 조회·이동

| 명령어 | 설명 | 예시 |
|--------|------|------|
| **ls** | 디렉터리 내용 나열 | `ls -l` (자세히), `ls -a` (숨김 포함) |
| **pwd** | 현재 작업 디렉터리 경로 출력 | `pwd` |
| **cd** | 디렉터리 이동 | `cd /tmp`, `cd ..`, `cd ~` |
| **cat** | 파일 내용 출력(연결) | `cat file.txt` |
| **more** | 파일 내용을 화면 단위로 출력 | `more file.txt` |
| **head** | 파일 앞부분 N줄 출력 | `head -5 file.txt` |
| **tail** | 파일 끝부분 N줄 출력 | `tail -10 file.txt` |

- `ls -l`: 권한, 링크 수, 소유자, 크기, 수정일 등 자세히 표시
- `ls -a`: `.`으로 시작하는 숨김 파일·디렉터리 포함
- `cd ~`: 홈 디렉터리로 이동
- `tail -f 파일`: 실시간으로 추가되는 내용 계속 출력(로그 확인 시 사용)

---

## 2. 파일·디렉터리 생성·삭제·복사·이동

| 명령어 | 설명 | 예시 |
|--------|------|------|
| **touch** | 빈 파일 생성 또는 수정 시간 갱신 | `touch a.txt` |
| **mkdir** | 디렉터리 생성 | `mkdir dir`, `mkdir -p a/b/c` |
| **rm** | 파일·디렉터리 삭제 | `rm file`, `rm -r dir` |
| **cp** | 복사 | `cp a b`, `cp -r dir1 dir2` |
| **mv** | 이동(이름 변경) | `mv old new` |

- `mkdir -p`: 상위 경로 없으면 함께 생성
- `rm -r`: 디렉터리와 그 안 내용까지 삭제
- `rm -f`: 확인 없이 강제 삭제
- `cp -r`: 디렉터리 복사

---

## 3. 권한(Permission)

| 명령어 | 설명 | 예시 |
|--------|------|------|
| **chmod** | 접근 권한 변경 | `chmod 755 file`, `chmod u+x file` |
| **chown** | 소유자 변경 | `chown user file` |
| **chgrp** | 소유 그룹 변경 | `chgrp group file` |

### 권한 표기

- **r(4)** 읽기, **w(2)** 쓰기, **x(1)** 실행
- **숫자**: 7=rwx, 6=rw-, 5=r-x, 4=r--, 0=---
- **755**: 소유자 rwx, 그룹 r-x, 기타 r-x
- **chmod u+x**: 소유자에게 실행 권한 추가

---

## 4. 프로세스·시스템

| 명령어 | 설명 | 예시 |
|--------|------|------|
| **ps** | 프로세스 목록 출력 | `ps -ef`, `ps aux` |
| **kill** | 프로세스 종료 | `kill 1234`, `kill -9 1234` |
| **top** | 실시간 프로세스·리소스 모니터링 | `top` |
| **jobs** | 백그라운드·중지된 작업 목록 | `jobs` |
| **fg / bg** | 포그라운드/백그라운드로 가져오기 | `fg %1`, `bg %1` |

- `kill -9`: 강제 종료(SIGKILL)
- `ps -ef`: 전체 프로세스 목록(UID, PID, PPID 등)

---

## 5. 검색·필터

| 명령어 | 설명 | 예시 |
|--------|------|------|
| **find** | 경로 아래에서 파일·디렉터리 검색 | `find . -name "*.c"` |
| **grep** | 텍스트에서 패턴 검색 | `grep "hello" file.txt` |
| **which** | 명령어 실행 파일 경로 출력 | `which ls` |
| **wc** | 줄·단어·바이트 수 출력 | `wc -l file.txt` |

- `find . -name "*.c"`: 현재 디렉터리 아래에서 이름이 `*.c`인 파일 검색
- `grep -r "패턴" dir`: 디렉터리 내 파일들에서 재귀 검색
- `wc -l`: 줄 수만 출력

---

## 6. 리다이렉션·파이프

| 기호/명령 | 설명 | 예시 |
|-----------|------|------|
| **>** | 표준 출력을 파일로 덮어쓰기 | `ls > out.txt` |
| **>>** | 표준 출력을 파일 끝에 추가 | `echo "a" >> out.txt` |
| **<** | 파일 내용을 표준 입력으로 | `program < in.txt` |
| **\|** | 앞 명령 출력을 뒤 명령 입력으로 | `ls \| grep ".c"` |

- `2>`: 표준 에러를 파일로
- `2>&1`: 표준 에러를 표준 출력과 같은 곳으로

---

## 7. 기타 자주 나오는 명령

| 명령어 | 설명 |
|--------|------|
| **echo** | 문자열 출력 (`echo $HOME` → 환경 변수 값) |
| **env** | 환경 변수 목록 출력 |
| **df** | 디스크 사용량(파일시스템 단위) |
| **du** | 디렉터리·파일 사용량 |
| **ln** | 링크 생성 (`ln -s target link` → 심볼릭 링크) |
| **uname** | 시스템 정보 (`uname -a`) |
| **who / whoami** | 로그인 사용자 / 현재 사용자 이름 |
| **diff** | 두 파일 차이 비교 |
