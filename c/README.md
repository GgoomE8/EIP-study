# C 언어 문제풀이

## 실행할 파일 하나만 실행 (권장)

문제 풀이 파일이 여러 개여도 **지정한 파일 하나만** 컴파일 후 실행합니다.

```bash
./run.sh 20260211_1.c
```

실행이 끝나면 생성된 실행 파일은 자동으로 삭제됩니다.

## 기타

### Makefile (main.c 전용)
```bash
make          # 컴파일
make run      # 컴파일 후 실행
make clean    # 실행 파일 삭제
```

### 직접 컴파일
```bash
gcc -Wall -Wextra -std=c11 -o 20260211_1 20260211_1.c
./20260211_1
```
