# 정보처리기사 문제풀이

각 언어별로 문제풀이를 할 수 있는 환경이 구성되어 있습니다.

## 디렉토리 구조

```
정보처리기사/
├── c/              # C 언어 문제풀이
├── java/           # Java 문제풀이
├── python/         # Python 문제풀이
├── 문제풀이/       # 맞춘 풀이 보관 (c, java, python)
├── 오답노트/       # 틀린 풀이 보관·재풀이 (c, java, python)
└── AGENT_RULES.md  # 파일 명명·실행·이동 규칙 (에이전트용)
```

## 사용 방법

각 언어별 디렉토리의 README.md를 참고하세요.
