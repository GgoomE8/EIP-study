# 에이전트 작업 규칙 (정보처리기사 문제풀이)

에이전트가 항상 읽고 실행·이동할 수 있도록 아래 규칙을 따릅니다.

---

## 1. 파일 명명 규칙

- **형식**: `yyyyMMdd_{numbering}`
  - 예: `20260211_1`, `20260211_2`
- **오늘 날짜**: 작업일 기준 `yyyyMMdd` (예: 20260211)
- **번호**: 같은 날짜에 이미 있는 번호가 있으면 **+1**해서 생성
  - 예: `20260211_1` 있음 → 다음 파일은 `20260211_2`
- **언어별 파일명**:
  - **C**: `c/20260211_1.c`, `c/20260211_2.c` …
  - **Java**: 클래스명이 숫자로 시작할 수 없으므로 `Q` 접두사 사용 → `java/Q20260211_1.java`, `java/Q20260211_2.java` …
  - **Python**: `python/20260211_1.py`, `python/20260211_2.py` …

---

## 2. 문제 문구 위치

- **항상 파일 최상단에 주석으로** 문제 내용을 둡니다.
- C/Java: `/* ... */` 블록 주석
- Python: `#` 줄 주석

---

## 2-1. 출력 설명 시 반드시 구분할 것

- **숫자 결과(합·곱 등)**를 원하면 문구에 **"합의 값"**, **"곱의 값"**처럼 명시한다.
  - 예: `출력: 합의 값 (정수)` → 1, 2 입력 시 `3` 출력
- **문자열로 식을 보여주는 것**을 원하면 **"A+B 형태"**, **"식 문자열"** 등으로 명시한다.
  - 예: `출력: "A+B" 형태 문자열` → 1, 2 입력 시 `1+2` 출력
- 문제에 그냥 **"A+B"**만 쓰면 해석이 갈리므로, 위 둘 중 하나로 반드시 구분해서 쓴다.

---

## 3. 실행 경로 (에이전트가 실행할 때)

- **C**: `c/run.sh 20260211_1.c` (프로젝트 루트 기준 `c/` 폴더에서)
- **Java**: `java/run.sh Q20260211_1.java` 또는 `java/run.sh Q20260211_1`
- **Python**: `python3 python/20260211_1.py` 또는 해당 폴더에서 `python3 20260211_1.py`

---

## 4. 풀이 결과에 따른 파일 이동

| 결과 | 이동 경로 |
|------|-----------|
| **맞음** | `./문제풀이/{언어}/` |
| **틀림** | `./오답노트/{언어}/` |

- **언어** 값: `c` | `java` | `python`
- 예: 맞춘 C 풀이 → `문제풀이/c/`, 틀린 Java 풀이 → `오답노트/java/`
- 오답노트로 옮긴 파일은 다시 풀 수 있도록 그대로 두거나, 수정 후 다시 `c/`, `java/`, `python/`에서 실행할 수 있음.

## 6. "다음 문제 언어별로 줘" 요청 시

- **생성 경로 (항상 작업 폴더)**  
  - C: `c/yyyyMMdd_N.c`  
  - Java: `java/QyyyyMMdd_N.java`  
  - Python: `python/yyyyMMdd_N.py`  
  - 여기서 `yyyyMMdd` = 오늘 날짜, `N` = 그날짜 기준 기존 번호 있으면 **최대 번호 + 1**, 없으면 **1**.

- **번호 정하는 방법**  
  - `c/`, `java/`, `python/`, `문제풀이/c`, `문제풀이/java`, `문제풀이/python`, `오답노트/c`, `오답노트/java`, `오답노트/python` 안의  
    `yyyyMMdd_*.c`, `QyyyyMMdd_*.java`, `yyyyMMdd_*.py` 파일을 보고, 같은 날짜 중 **가장 큰 번호 + 1**을 사용.

- **파일 내용 (반드시 지킬 것)**  
  - **풀이를 넣지 않는다.** 사용자가 직접 풀어야 함.  
  - **각 언어의 main 파일을 읽어서** 그 구조를 그대로 쓰고, **최상단에만 문제 문구 주석**을 붙인 **빈 템플릿**으로 생성한다.  
  - 참조할 main 파일 경로:  
    - C: `c/main.c`  
    - Java: `java/Main.java` (단, 클래스명은 파일명에 맞게 `QyyyyMMdd_N`으로 변경)  
    - Python: `python/main.py`  
  - main과 동일하게 `// 문제풀이 코드 작성` 또는 `# 문제풀이 코드 작성` 자리는 비워 두고, 실행 가능한 골격만 둔다.

- **어떻게 줄지**  
  - 위 경로에 파일 3개를 **빈 템플릿으로만** 생성해서 준다. 문제 문구는 사용자가 지정하면 그대로, 없으면 기본 문구를 주석으로 넣는다.
